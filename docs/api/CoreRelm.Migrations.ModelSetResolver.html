ResolveSetWithDiagnostics(ModelSetsFile, string)

Resolves a model set definition by name from the specified model sets file and returns the resolved set along with detailed diagnostics.

public (ResolvedModelSet Resolved, ResolvedModelSetDiagnostics Diagnostics) ResolveSetWithDiagnostics(ModelSetsFile file, string setName)

Parameters

file ModelSetsFile

The model sets file containing the definitions to resolve.

setName string

The name of the model set to resolve. Must match a set defined in the provided file.

Returns

(ResolvedModelSet Resolved, ResolvedModelSetDiagnostics Diagnostics)

A tuple containing the resolved model set and a diagnostics object with information about the resolution process, including any errors or warnings encountered.

Remarks

The diagnostics object provides detailed information about the resolution process, such as missing types, attribute validation errors, and counts of included or excluded types. This method does not throw for missing or invalid attributes; instead, such issues are reported in the diagnostics.

Exceptions

ArgumentException

Thrown if a model set with the specified name does not exist in the provided file.
