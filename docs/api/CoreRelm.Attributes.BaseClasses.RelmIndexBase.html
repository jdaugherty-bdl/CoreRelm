Remarks

Apply this attribute to a class, struct, or property to define custom index settings for use by an ORM or database schema generator. Multiple instances can be applied to specify multiple indexes. Not all options may be supported by all database providers; unsupported options may be ignored.

public RelmIndexBase(string[] indexedPropertyNames = null, string indexName = null, Indexes.IndexType indexType = IndexType.None, int keyBlockSize = -1, string parserName = null, string comment = null, Indexes.Visibility visibility = Visibility.None, string engineAttribute = null, string secondaryEngineAttribute = null, Indexes.Algorithm algorithmType = Algorithm.None, Indexes.LockOption lockOption = LockOption.None)

Parameters

indexedPropertyNames string[]
indexName string

The name of the index. If null or empty, a default name may be generated by the data provider.

indexType Indexes.IndexType

The type of index to create, such as a standard or full-text index. Controls the storage and behavior of the index.

keyBlockSize int

The key block size for the index, in bytes. Use -1 to indicate the default size. Must be a positive integer or -1.

parserName string

The name of the parser to use for the index, if applicable. Used for specialized index types such as full-text indexes.

comment string

A comment or description for the index. This value may be stored in the database metadata.

visibility Indexes.Visibility

The visibility of the index, such as visible or invisible. Controls whether the index is used by the query optimizer.

engineAttribute string

The name of the storage engine attribute to use for the index, if supported by the database engine.

secondaryEngineAttribute string

The name of the secondary engine attribute to use for the index, if supported by the database engine.

algorithmType Indexes.Algorithm

The algorithm to use for the index, such as B-tree or hash. Determines the internal structure of the index.

lockOption Indexes.LockOption

The lock option to use when creating or modifying the index. Controls concurrency behavior during index operations.

Remarks

Apply this attribute to a class, struct, or property to define custom index settings for use by an ORM or database schema generator. Multiple instances can be applied to specify multiple indexes. Not all options may be supported by all database providers; unsupported options may be ignored.
