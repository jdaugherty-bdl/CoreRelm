Remarks

Use this class to specify index key details when defining indexes, including support for partial indexes or computed expressions. The combination of properties allows for flexible index definitions, such as descending order or partial column indexing.

public RelmIndexColumnBase(string columnName, int length = -1, string expression = null, bool isDescending = false, int order = 0)

Parameters

columnName string

The name of the column used as the index key. Cannot be null or empty.

length int

The maximum length, in characters, to use for the index key. If null, the full column value is used.

expression string

An optional expression used to define the index key. If specified, the expression is used instead of the column value.

isDescending bool

Indicates whether the index key is sorted in descending order. Set to true for descending; otherwise, false.

order int

The zero-based position of the item within a collection or sequence.
