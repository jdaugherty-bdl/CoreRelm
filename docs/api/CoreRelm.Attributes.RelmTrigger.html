Remarks

Apply this attribute to properties or structs to define custom trigger behavior in Relm-based applications. Multiple triggers can be configured by using this attribute with different parameters. The attribute does not enforce trigger execution; it serves as metadata for frameworks or tools that interpret trigger definitions.

Parameters

triggerTime Triggers.TriggerTime

The time at which the trigger is executed. Determines whether the trigger occurs before or after a specified event.

triggerEvent Triggers.TriggerEvent

The event that activates the trigger. Indicates the action or operation that causes the trigger to run.

triggerBody string

The body or logic of the trigger. Represents the code or expression to be executed when the trigger is activated. Can be null if no body is specified.

triggerName string

The name of the trigger associated with this instance.

triggerOrder Triggers.TriggerOrdering

The order in which the trigger is executed relative to other triggers. Optional; can be null if no specific order is required.

otherTriggerName string

The name of another trigger to reference or relate to. Optional; can be null if not applicable.

Exceptions

ArgumentNullException

Thrown if triggerBody is null.
