Remarks

Use this when no correlation key is required. All configuration options are identical to the generic version, except no index key is supplied.

Constructors

RelmIndexNamed(string[], string, IndexType, int, string, string, Visibility, string, string, Algorithm, LockOption)

Initializes a new instance of the non-keyed RelmIndex with required indexed property names and optional index configuration settings.

public RelmIndexNamed(string[] indexedPropertyNames, string indexName = null, Indexes.IndexType indexType = IndexType.None, int keyBlockSize = -1, string parserName = null, string comment = null, Indexes.Visibility visibility = Visibility.None, string engineAttribute = null, string secondaryEngineAttribute = null, Indexes.Algorithm algorithmType = Algorithm.None, Indexes.LockOption lockOption = LockOption.None)

Parameters

indexedPropertyNames string[]

Required array of property names to be included in the index. Cannot be null or empty.

indexName string

The name of the index. If null, a default name may be used.

indexType Indexes.IndexType

The type of index to create, such as primary or secondary.

keyBlockSize int

The size, in bytes, of the key block for the index. Specify -1 to use the default size.

parserName string

The name of the parser to use for the index, or null to use the default parser.

comment string

An optional comment describing the index.

visibility Indexes.Visibility

Specifies the visibility of the index, such as public or private.

engineAttribute string

An optional engine-specific attribute for the index.

secondaryEngineAttribute string

An optional secondary engine-specific attribute for the index.

algorithmType Indexes.Algorithm

The algorithm to use for the index, or Algorithm.None to use the default.

lockOption Indexes.LockOption

The locking option to use for the index, or LockOption.None for the default behavior.
